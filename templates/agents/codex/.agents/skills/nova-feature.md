# Skill: nova-feature (SDD)

Para iniciar uma nova funcionalidade:

1. Leia `methodology/sdd.md` e o bundle `knowledge/`.
2. Crie `specs/NNNN-<slug>/spec.md` (número sequencial) a partir de
   `specs/0001-exemplo/spec.md`.
3. Preencha Contexto, Objetivo, Escopo e Critérios de aceitação; linke conceitos
   OKF (`/knowledge/concepts/...`).
4. Implemente só após a spec estar clara. Registre conhecimento durável em
   `knowledge/concepts/` e em `knowledge/log.md`.
