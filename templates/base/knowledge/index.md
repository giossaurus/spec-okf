# Bundle de Conhecimento (OKF)

> Este diretório é um bundle **Open Knowledge Format v0.1**. Cada arquivo em
> `concepts/` é um conceito (markdown + frontmatter YAML). Este `index.md` é a
> porta de entrada para divulgação progressiva — não tem frontmatter.

## Conceitos

* [Exemplo](concepts/exemplo.md) - conceito de exemplo; substitua pelos seus.

## Convenções

* Campo `type` é obrigatório em cada conceito.
* Links entre conceitos usam caminho bundle-relativo: `/concepts/<arquivo>.md`.
* Registre mudanças relevantes em [log](log.md).
